# 여행 비용 공유 · Travel Split

여행 비용을 기록하고 N분의 1로 정산하는 **단일 파일 웹앱**. 설치·서버 없이 `index.html`만 열면 됩니다.

## 사용법
1. `travel-expense/index.html`을 브라우저로 엽니다. (더블클릭 또는 호스팅)
2. **새 여행** → 여행 이름 / 기준 통화 / 참여자 입력.
3. **비용 추가**:
   - 🔎 검색창에 `항공권`, `호텔`, `라멘` 등을 입력 → 항목 선택 시 분류·금액·통화 **자동 입력** (수정 가능).
   - 또는 내용·금액·통화·결제자·날짜·분담 인원을 **수동 입력**.
4. 우측에서 총 지출, 인원별 잔액, **누가 누구에게 보낼지 송금 정산**을 실시간 확인.

## 핵심 기능
- **DB 정리**: 여러 여행 / 참여자 / 비용 내역을 브라우저 `localStorage`에 저장.
- **동시 작업**: 같은 브라우저의 여러 탭은 `BroadcastChannel`로 실시간 동기화.
- **공유**: `🔗 공유 링크 복사`(데이터 인코딩 URL) 또는 `JSON 내보내기/가져오기`로 다른 사람과 공유.
- **가격 검색 자동 입력**: 여행 항목 카탈로그 검색 → 클릭 자동 채움.
- **수동 입력**: 모든 항목 직접 입력/수정 가능.
- **환율 자동 변환**: [open.er-api.com](https://open.er-api.com) 무료 환율로 외화를 기준 통화로 환산 (캐시, 오프라인 시 1:1 폴백).

## 한계 / 참고
- 백엔드가 없으므로 **다른 기기와의 실시간 동시 편집은 안 됩니다.** 진짜 멀티유저 실시간이 필요하면 Supabase 같은 백엔드로 확장하세요.
